git init: dùng để khởi tạo git. Ban đầu các file được tạo ra (chưa init) thì đang ở local, sau khi dùng lệnh này thì sẽ khởi tạo git để di chuyển các file sang vùng working directory;
git add file_name: thêm 1 file được xác định vào vùng stg;
git add .: thêm tất cả các file vào vùng stg;
git commit -m"message": thực hiện commit, snapshot stg vào repo, files vẫn còn ở working;
